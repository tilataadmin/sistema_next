Guía de Variables
Variables de texto corto:
  • $$contratante$$
  • $$contratista$$
  • $$salario$$
  • $$cargo$$
Variables de texto largo:
  • $$objeto_large$$ o $$objeto_texto$$
  • $$condiciones_large$$
  • $$descripcion_texto$$
  • O cualquier variable con: condicion, descripcion, detalle
Tip: Las variables que terminan en _large o _texto, o contienen palabras como "condicion", "descripcion" o "detalle" se mostrarán como campos de texto largo (textarea) al generar el contrato.
Las variables deben estar en formato $$nombreVariable$$
