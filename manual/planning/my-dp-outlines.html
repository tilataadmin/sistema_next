¿Qué es el esquema bi-anual?

El esquema bi-anual es el documento de planificación de largo plazo del Programa del Diploma (PD). Cubre los dos años completos del programa (10° y 11°) para una asignatura y nivel específico.

A diferencia de los planeadores trimestrales —que se crean cada trimestre— el esquema bi-anual es un documento vivo: existe una sola versión por asignatura y nivel, y se actualiza a lo largo del tiempo sin necesidad de recrearlo cada año.

¿Qué son los niveles SL y HL?
El IB ofrece dos niveles para cada asignatura:
SL Nivel Medio — menor carga horaria y menor profundidad.
HL Nivel Superior — mayor carga horaria y mayor profundidad.
Cada nivel genera un esquema independiente.

¿Para qué sirve esta pantalla?

En Mis esquemas bi-anuales PD puede:

  • Ver todos los esquemas bi-anuales de sus asignaturas.
  • Crear un esquema nuevo para una asignatura y nivel que aún no lo tenga.
  • Acceder a cualquier esquema existente con un clic para revisarlo o editarlo.
Esta pantalla es personal.
Solo aparecen las asignaturas que tiene asignadas. Si una asignatura no aparece en la lista de creación, es porque no tiene asignación activa para ese período.

Cómo crear un esquema nuevo

1
Seleccione la asignatura en el menú desplegable. Solo aparecen las asignaturas con asignación activa en el año vigente.
2
Seleccione el nivel: SL o HL. Si dicta la misma asignatura en ambos niveles, debe crear un esquema para cada uno por separado.
3
Haga clic en "Crear". El sistema verifica que no exista ya un esquema para esa combinación. Si ya existe, le mostrará un aviso con un enlace directo al esquema existente.
4
El sistema abre automáticamente el formulario del esquema recién creado, listo para que empiece a diligenciar.
Consejo: Cree primero el esquema bi-anual antes de crear los planeadores trimestrales. El sistema vincula automáticamente los planeadores al esquema cuando los crea.

Ver y abrir esquemas existentes

La tabla inferior muestra todos los esquemas de sus asignaturas. Cada fila incluye:

  • Asignatura y nivel (SL o HL).
  • Completado por — quién figura como responsable del esquema.
  • Última edición — fecha y hora del último cambio guardado.

Haga clic en cualquier fila para abrir el esquema y editarlo o consultarlo.

Esquemas creados por otros docentes:
Si un colega de la misma asignatura ya creó el esquema, también aparecerá en su listado con una nota indicando quién lo creó. Puede abrirlo, pero solo podrá editarlo si tiene asignación activa para esa asignatura o si es coordinador del área.

Solución de problemas frecuentes

No aparece ninguna asignatura en el menú de creación

Esto ocurre cuando no tiene asignaciones académicas activas para el año vigente. Contacte al coordinador de área o al administrador del sistema para verificar su carga académica.

Aparece el aviso "Ya existe este esquema"

El esquema para esa asignatura y nivel ya fue creado. Use el enlace "Abrir planeador" que aparece en el aviso para acceder directamente al documento existente.

El botón "Crear" no se activa

Debe seleccionar tanto la asignatura como el nivel para que el botón se habilite. Revise que ambos campos estén seleccionados.

No veo el esquema que acabo de crear

El sistema lo redirige automáticamente al formulario del esquema recién creado. Cuando regrese a esta pantalla, el esquema aparecerá en la tabla.

Preguntas frecuentes

¿Tengo que crear un esquema nuevo cada año académico?
No. El esquema bi-anual es un documento permanente que se actualiza con el tiempo. No tiene año académico asignado — pertenece a la asignatura y nivel de forma indefinida.
¿Quién puede editar mi esquema?
Usted (como docente con asignación activa), el coordinador de su área, el director del programa PD y el director de sección. El resto de usuarios pueden consultarlo pero no modificarlo.
¿Puedo crear un esquema para una asignatura que no dicto este año?
No directamente desde esta pantalla, ya que el menú solo muestra sus asignaciones del año vigente. Si necesita crear un esquema para una asignatura histórica, contáctese con el administrador.
¿Qué pasa si dicto la misma asignatura en SL y HL?
Debe crear dos esquemas independientes: uno para SL y otro para HL. Ambos aparecerán en su listado.
