Estructura de Autoevaluación

Si tienes dudas o encuentras algún problema,
puedes reportarlo usando el botón de abajo.

¿Qué es la Estructura de Autoevaluación?

Esta función permite configurar todo lo que se va a evaluar en la autoevaluación institucional. Es como armar el "esqueleto" o el mapa de la evaluación antes de que alguien empiece a calificar.

La estructura se organiza de forma jerárquica, como un árbol: primero las grandes áreas de gestión, dentro de cada área los procesos, dentro de cada proceso los componentes específicos que se evalúan, y para cada componente se definen las rúbricas que describen qué significa obtener un 1, 2, 3 o 4.

Adicionalmente, puedes definir fuentes (marcos de referencia como la Guía 34) y resultados esperados para cada proceso.

IMPORTANTE: Esta función es de configuración inicial. Debe completarse antes de crear evaluaciones. Cambiar la estructura después de que existan evaluaciones en curso puede afectar los datos.

¿Cómo se organiza la estructura?

La estructura se organiza en niveles jerárquicos. Cada nivel contiene al siguiente:

Fuentes — Marcos de referencia (ej: "Guía 34 del MEN")
Áreas de Gestión — Grandes áreas (ej: "Gestión Directiva")
Procesos — Procesos dentro de cada área (ej: "Direccionamiento estratégico")
Componentes — Aspectos puntuales a evaluar (ej: "Misión, visión y principios")
Rúbricas — Descripción de cada nivel 1 a 4 para el componente
Resultados Esperados — Metas asociadas a cada proceso
Ejemplo práctico: La fuente "Guía 34" contiene el área "Gestión Directiva", que tiene el proceso "Direccionamiento estratégico", que incluye el componente "Misión, visión y principios". Este componente tiene 4 rúbricas que describen qué significa cada nivel de valoración.

Características principales

  • Gestión de Fuentes: Crear y organizar los marcos de referencia que agrupan las áreas
  • Áreas de Gestión: Crear, editar y ordenar las grandes áreas de evaluación
  • Procesos: Definir los procesos dentro de cada área, con descripción y orden
  • Componentes: Crear los aspectos específicos que se califican en cada proceso
  • Rúbricas: Describir en detalle qué significa cada nivel (1 a 4) para cada componente
  • Resultados Esperados: Definir metas o indicadores asociados a cada proceso
  • Ordenamiento: Todos los elementos se pueden ordenar numéricamente para controlar el orden de presentación
  • Indicador visual: Los componentes muestran si sus rúbricas están completas o pendientes

Antes de comenzar

Requisitos previos:
  • Tener asignado el permiso "Configurar estructura de autoevaluación"
  • Conocer la estructura de áreas y procesos que la institución desea evaluar
  • Tener a mano las descripciones de los componentes y sus rúbricas (idealmente de la Guía 34 o del marco institucional)
Advertencia: Se recomienda completar toda la estructura antes de crear la primera evaluación. Aunque se puede modificar después, los cambios no se reflejan retroactivamente en evaluaciones que ya están en curso.
Nota: Las fuentes son opcionales. Si la institución solo utiliza un marco de referencia (la Guía 34, por ejemplo), puede crear una sola fuente y asociar todas las áreas a ella. También es válido no asociar fuentes a las áreas.

Guía paso a paso

1 Gestionar las Fuentes (opcional)

Las fuentes son los marcos de referencia bajo los cuales se organizan las áreas. Por ejemplo, "Guía 34 del MEN" o "PEI Institucional".

Para crear una fuente:

  1. Haz clic en el botón "Fuentes" ubicado en la parte superior derecha de la página.
  2. Se abrirá una ventana con un formulario sencillo.
  3. Escribe el nombre de la fuente (obligatorio).
  4. Opcionalmente, agrega una descripción y un número de orden.
  5. Haz clic en el botón "+" para guardar.

Las fuentes creadas aparecerán en una lista debajo del formulario. Desde ahí puedes editarlas o eliminarlas.

Nota: Las fuentes estarán disponibles como opción al crear áreas, procesos y componentes. Es una forma de clasificar de dónde proviene cada elemento de la estructura.

2 Crear las Áreas de Gestión

Las áreas son las grandes categorías de evaluación. Según la Guía 34, son cuatro: Directiva, Académica, Administrativa y Financiera, y Comunitaria. Tu institución puede definir las que necesite.

Para crear un área:

  1. Haz clic en el botón "Nueva Área".
  2. En la ventana que se abre, escribe el nombre del área (obligatorio).
  3. Opcionalmente, agrega una descripción, selecciona una fuente y asigna un número de orden.
  4. Haz clic en "Guardar".

Las áreas creadas aparecerán como tarjetas en la parte superior de la página. Haz clic en una tarjeta para seleccionarla y ver sus procesos.

Ejemplo: Crea las áreas: "Gestión Directiva" (orden 1), "Gestión Académica" (orden 2), "Gestión Administrativa y Financiera" (orden 3), "Gestión de la Comunidad" (orden 4).

3 Crear los Procesos dentro de cada Área

Cada área contiene procesos. Para agregar procesos, primero debes seleccionar un área haciendo clic en su tarjeta.

Para crear un proceso:

  1. Selecciona el área haciendo clic en su tarjeta.
  2. Debajo aparecerá la sección "Procesos de: [Nombre del Área]".
  3. Haz clic en "Nuevo Proceso".
  4. Escribe el nombre del proceso (obligatorio).
  5. Opcionalmente, agrega descripción, fuente y orden.
  6. Haz clic en "Guardar".

Los procesos aparecerán como acordeones (secciones expandibles). Haz clic en el nombre del proceso para expandirlo y ver sus componentes y resultados esperados.

Ejemplo: Dentro de "Gestión Directiva" podrías crear los procesos: "Direccionamiento estratégico y horizonte institucional", "Gestión estratégica", "Gobierno escolar", etc.

4 Crear los Componentes de cada Proceso

Los componentes son los aspectos puntuales que se califican. Son el corazón de la evaluación.

Para crear un componente:

  1. Expande el proceso haciendo clic en su nombre.
  2. Dentro del proceso, haz clic en "Nuevo Componente".
  3. Escribe el nombre del componente (obligatorio).
  4. Opcionalmente, agrega una definición que explique el alcance del componente.
  5. Selecciona fuente y orden si lo deseas.
  6. Haz clic en "Guardar".

Los componentes aparecerán dentro del acordeón del proceso. Cada componente muestra un indicador que dice si sus rúbricas están completas o pendientes.

Importante: Un componente no puede ser evaluado hasta que tenga las 4 rúbricas completas (una por cada nivel). El indicador visual te muestra el estado.

5 Definir las Rúbricas de cada Componente

Las rúbricas describen qué significa cada nivel de valoración para un componente específico. Son la guía que usarán los evaluadores al momento de calificar.

Para definir las rúbricas:

  1. Ubica el componente dentro de su proceso.
  2. Haz clic en el botón "Rúbricas" del componente (ícono de checklist).
  3. Se abrirá una ventana con 4 campos de texto, uno por cada nivel:

1 Existencia: Describe una situación incipiente, desorganizada, sin planeación.

2 Pertinencia: Describe un principio de planeación y articulación de esfuerzos.

3 Apropiación: Describe acciones consolidadas y conocidas por la comunidad.

4 Mejoramiento Continuo: Describe evaluación y mejora permanente con resultados sostenibles.

  1. Escribe una descripción clara para cada uno de los 4 niveles.
  2. Haz clic en "Guardar Rúbricas".
CRÍTICO: Los 4 niveles son obligatorios. Si dejas alguno vacío, el componente quedará marcado como "Rúbricas pendientes" y no podrá ser evaluado.
Consejo: Redacta las rúbricas en tercera persona y de forma progresiva. Cada nivel debe representar un avance claro respecto al anterior. Los evaluadores usarán estas descripciones para decidir su calificación.

6 Definir Resultados Esperados (opcional)

Los resultados esperados son metas o indicadores que la institución desea lograr en cada proceso. Son útiles para orientar el plan de mejoramiento.

Para agregar un resultado esperado:

  1. Expande el proceso haciendo clic en su nombre.
  2. Busca la sección de resultados esperados dentro del proceso.
  3. Haz clic en "Nuevo Resultado".
  4. Escribe la descripción del resultado (obligatorio).
  5. Asigna un orden si deseas controlar la secuencia.
  6. Haz clic en "Guardar".
Nota: Los resultados esperados son opcionales y no afectan la evaluación directamente. Son un complemento para documentar las metas del proceso.

7 Editar o eliminar elementos

Todos los elementos de la estructura (fuentes, áreas, procesos, componentes, rúbricas y resultados) se pueden editar o eliminar.

Para editar:

  • Las áreas tienen un botón "Editar Área" que aparece al seleccionarla.
  • Los procesos, componentes y resultados tienen botones de edición (ícono de lápiz) visibles al expandir el acordeón.
  • Las rúbricas se editan desde el mismo botón "Rúbricas" del componente.

Para eliminar:

  • Usa el botón de eliminar (ícono de papelera) de cada elemento.
  • El sistema pedirá confirmación antes de borrar.
Precaución: Eliminar un área borrará también todos sus procesos, componentes y rúbricas asociados. Lo mismo ocurre al eliminar un proceso: se borrarán sus componentes. Usa esta opción con cuidado.

Buenas prácticas

  • Completa la estructura antes de crear evaluaciones. Cambios posteriores pueden generar inconsistencias
  • Usa el campo de orden para controlar cómo se presentan los elementos. Un orden claro facilita la evaluación
  • Redacta rúbricas claras y progresivas. Cada nivel debe ser distinguible del anterior
  • No dejes componentes sin rúbricas. Esos componentes no podrán evaluarse
  • Revisa todo antes de iniciar la evaluación. Una vez creada, los cambios en estructura no se reflejan automáticamente
  • Usa las fuentes para organizar si la estructura proviene de varios documentos o marcos normativos
  • Trabaja área por área para mantener el orden: primero crea todos los procesos de un área, luego todos los componentes, y finalmente todas las rúbricas

Solución de problemas comunes

No puedo ver los procesos de un área
  • Asegúrate de haber hecho clic en la tarjeta del área. La tarjeta seleccionada se resaltará visualmente.
  • Si el área no tiene procesos, verás un mensaje indicándolo con un botón para crear el primero.
El componente dice "Rúbricas pendientes"
  • Esto significa que aún no has definido los 4 niveles de la rúbrica para ese componente.
  • Haz clic en el botón "Rúbricas" del componente y completa los 4 campos.
  • Recuerda que los 4 niveles son obligatorios.
No aparece el botón "Nueva Área" o "Nuevo Proceso"
  • Verifica que tengas el permiso "Configurar estructura de autoevaluación" asignado.
  • Si no ves los botones, contacta al administrador del sistema.
Eliminé un área por error
  • Lamentablemente, la eliminación es permanente y no se puede deshacer.
  • Deberás recrear el área con todos sus procesos, componentes y rúbricas.
  • Por esto es importante confirmar siempre antes de eliminar.
Los elementos no aparecen en el orden que espero
  • Revisa el campo "Orden" de cada elemento. Los elementos se muestran de menor a mayor número.
  • Si dos elementos tienen el mismo orden, se mostrarán en el orden en que fueron creados.

Preguntas frecuentes

¿Puedo modificar la estructura después de crear una evaluación?

Sí, técnicamente puedes. Sin embargo, los cambios no se reflejan automáticamente en evaluaciones que ya están en curso. Lo ideal es completar la estructura antes de iniciar evaluaciones.

¿Cuántas áreas, procesos y componentes puedo crear?

No hay un límite fijo. Puedes crear tantos como necesite tu institución. Sin embargo, una estructura muy extensa puede hacer más largo el proceso de evaluación.

¿Es obligatorio usar fuentes?

No. Las fuentes son opcionales y sirven para clasificar de dónde proviene cada elemento. Puedes dejar el campo de fuente vacío al crear áreas, procesos o componentes.

¿Qué pasa si no completo las rúbricas de un componente?

El componente quedará marcado como "Rúbricas pendientes" y los evaluadores no podrán calificarlo. Aparecerá visualmente deshabilitado durante la evaluación.

¿Puedo reutilizar la misma estructura para varios años?

Sí. La estructura se define una vez y las evaluaciones anuales se crean sobre esa misma estructura. Solo necesitas ajustarla si cambian las áreas, procesos o componentes de un año a otro.

¿Qué son los resultados esperados y para qué sirven?

Son metas o indicadores que describes para cada proceso. No se califican directamente, pero sirven de referencia para el plan de mejoramiento y para documentar qué espera lograr la institución en cada proceso.

¿Puedo importar la estructura desde un archivo?

Actualmente no. La estructura se crea manualmente desde la interfaz. Sin embargo, una vez creada, se mantiene guardada permanentemente y no es necesario volver a ingresarla.

¿Quién puede configurar la estructura?

Solo los usuarios que tengan asignado el permiso "Configurar estructura de autoevaluación" o los administradores del sistema (Super Admin).

¿Encontraste un problema o tienes una sugerencia?

Si experimentaste un error, algo no funciona como esperas, o tienes una sugerencia para mejorar el sistema, repórtalo aquí.
