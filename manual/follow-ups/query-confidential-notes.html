Consultar Notas Confidenciales

Si tienes dudas o encuentras algún problema,
puedes reportarlo usando el botón de abajo.

¿Qué es Consultar Notas Confidenciales?

Esta función permite buscar y visualizar notas confidenciales registradas sobre estudiantes. Las notas confidenciales son observaciones, reflexiones o información sensible que requiere máxima confidencialidad y solo es accesible para personal específicamente autorizado.

A diferencia de los seguimientos regulares, las notas confidenciales NO se comparten con el equipo docente general. Son herramientas exclusivas para directivos, estrategas, equipo EAE y personal con permisos especiales, diseñadas para documentar información que por su naturaleza debe mantenerse en círculo restringido.

ADVERTENCIA CRÍTICA DE CONFIDENCIALIDAD:
  • Esta función maneja información ALTAMENTE SENSIBLE Y CONFIDENCIAL
  • El acceso está restringido a roles específicos con necesidad de conocer
  • Toda información consultada debe mantenerse en ESTRICTA CONFIDENCIALIDAD
  • NO compartir, discutir o mencionar contenido fuera del círculo autorizado
  • NO tomar capturas de pantalla sin autorización expresa
  • Violar la confidencialidad puede tener consecuencias legales y disciplinarias
Propósito principal: Facilitar la búsqueda y consulta de notas confidenciales, permitiendo:
  • Buscar notas por nombre de estudiante
  • Filtrar por rango de fechas
  • Ver notas propias o de todo el equipo (según permisos)
  • Consultar información sensible de forma segura
  • Mantener trazabilidad de quién registró cada nota
¿Quién puede usar esta función?
  • Super Administradores: Acceso completo a todas las notas
  • Rol "Estrategas": Acceso completo a todas las notas
  • Rol "Directivos docentes": Acceso completo a todas las notas
  • Rol "EAE": Acceso completo a todas las notas
  • Usuarios autorizados sin rol especial: Solo ven sus propias notas
Sistema de permisos por roles: El sistema determina automáticamente qué notas puedes ver:
  • Si tienes uno de los roles especiales: Verás TODAS las notas del sistema
  • Si NO tienes rol especial: Solo verás las notas que TÚ creaste
  • No hay opción manual para cambiar este comportamiento

Características principales

  • Solo lectura: Función exclusiva para consulta, no para crear o editar
  • Búsqueda por nombre: Encuentra notas de estudiantes específicos
  • Filtro por fechas: Rango "desde" y "hasta" para búsquedas temporales
  • Control de permisos por rol: Automático según tu nivel de autorización
  • Fotografías de estudiantes: Identificación visual rápida
  • Vista en tarjetas: Presentación clara y organizada
  • Fecha de registro: Saber cuándo se creó cada nota
  • Identificación del autor: Sistema registra quién creó cada nota (aunque no se muestra en esta vista)
  • Ordenamiento cronológico: Notas más recientes primero
  • Scroll interno: Notas largas con desplazamiento para mejor lectura
  • Diseño responsive: Funciona en computadores, tablets y móviles
  • Bordes rojos distintivos: Recordatorio visual del carácter confidencial

Antes de comenzar

Requisitos previos:
  • Tener el permiso "Consultar notas confidenciales" asignado
  • Pertenecer a uno de los roles autorizados o tener permiso especial
  • Comprender y aceptar la responsabilidad de confidencialidad
  • Conocer los protocolos institucionales de manejo de información sensible
Responsabilidad legal y ética:
  • Deber de confidencialidad: Toda información es estrictamente confidencial
  • Uso exclusivo autorizado: Solo para fines profesionales específicos
  • Prohibición de divulgación: No compartir bajo ninguna circunstancia
  • No almacenamiento externo: No copiar, imprimir o guardar fuera del sistema
  • Pantallas seguras: Asegurar que nadie no autorizado pueda ver la pantalla
  • Cierre de sesión: Cerrar siempre al terminar de consultar
¿Qué contienen las notas confidenciales? Pueden incluir (ejemplos NO exhaustivos):
  • Observaciones sobre situaciones familiares delicadas
  • Reflexiones sobre aspectos sensibles del estudiante
  • Información que no debe ser de conocimiento público
  • Contextos confidenciales necesarios para el acompañamiento
  • Notas de reuniones confidenciales con especialistas
Cuándo usar esta función:
  • Antes de reuniones importantes con padres o especialistas
  • Para recordar contextos sensibles antes de intervenciones
  • Durante coordinación confidencial entre directivos/EAE
  • Para verificar información registrada previamente
  • En casos que requieren revisar historial confidencial
Cuándo NO usar esta función:
  • Para crear nuevas notas (usa la función de creación correspondiente)
  • Si la información puede obtenerse de fuentes regulares no confidenciales
  • Para compartir con personas no autorizadas (NUNCA)
  • Si no tienes necesidad profesional específica de acceder

Guía paso a paso

1 Comprender tus permisos de visualización

Antes de buscar, es importante que entiendas qué notas podrás ver según tu rol.

Sistema automático de permisos:

El sistema verifica automáticamente tu rol al ingresar:

  • Si eres: Super Admin, Estrategas, Directivos docentes, o EAE
  • Verás: TODAS las notas confidenciales del sistema
  • Sin importar: Quién las creó o de qué estudiante son

PERO si NO tienes ninguno de esos roles:

  • Verás: SOLO las notas que TÚ creaste personalmente
  • NO verás: Notas creadas por otros usuarios
¿Cómo saber qué puedo ver?
  • El sistema NO muestra un mensaje indicándolo (es automático)
  • Si haces una búsqueda y obtienes pocos o ningún resultado, puede ser porque:
    • No tienes rol especial y solo ves tus propias notas
    • Realmente no hay notas que coincidan con tu búsqueda
  • Si tienes duda sobre tu nivel de acceso, consulta con el administrador

2 Realizar una búsqueda básica

La interfaz ofrece tres filtros que puedes usar individualmente o combinados.

Campos de búsqueda disponibles:

  • Estudiante: Buscar por nombre o apellido del estudiante
  • Desde: Fecha inicial del rango de búsqueda
  • Hasta: Fecha final del rango de búsqueda

Para realizar una búsqueda simple:

  1. Escribe el nombre o apellido del estudiante en el campo "Estudiante"
  2. Haz clic en el botón rojo "Buscar"
  3. El sistema mostrará las notas que coincidan con el nombre
Sobre la búsqueda por nombre:
  • No distingue mayúsculas de minúsculas
  • Busca coincidencias parciales (por ejemplo, "Juan" encontrará "Juan Carlos")
  • Busca en nombres y apellidos
  • No necesitas escribir el nombre completo
Ejemplos de búsqueda por nombre:
  • Escribes: "García" → Encontrará: María García, Pedro García López, etc.
  • Escribes: "Ana" → Encontrará: Ana María, Juana, Mariana, etc.
  • Escribes: "Ro" → Encontrará: Rodríguez, Romero, Roberto, etc.

3 Usar filtros de fecha

Los filtros de fecha te permiten buscar notas creadas en un período específico.

Para filtrar por fechas:

  1. Haz clic en el campo "Desde" y selecciona la fecha inicial
  2. Haz clic en el campo "Hasta" y selecciona la fecha final
  3. Opcionalmente, agrega un nombre de estudiante
  4. Haz clic en "Buscar"
Funcionamiento de los filtros de fecha:
  • Solo "Desde": Mostrará notas desde esa fecha en adelante
  • Solo "Hasta": Mostrará notas desde el inicio hasta esa fecha
  • Ambos campos: Mostrará notas dentro del rango exacto
  • Ninguno: No aplicará filtro de fecha (mostrará todas según permisos)
Ejemplos prácticos:
  • Notas de este mes: "Desde" = primer día del mes, "Hasta" = hoy
  • Notas de la semana pasada: "Desde" = lunes pasado, "Hasta" = domingo pasado
  • Notas recientes de un estudiante: Nombre + "Desde" = hace 30 días
  • Historial completo: Dejar ambos campos vacíos
Importante sobre fechas:
  • Las fechas se basan en cuando se CREÓ la nota, no en eventos mencionados
  • El formato de fecha puede variar según tu navegador y configuración regional
  • El campo "Hasta" incluye todo el día seleccionado hasta las 23:59

4 Combinar filtros para búsquedas precisas

La herramienta más poderosa es combinar nombre de estudiante con rango de fechas.

Para una búsqueda combinada:

  1. Escribe el nombre del estudiante en "Estudiante"
  2. Selecciona la fecha "Desde"
  3. Selecciona la fecha "Hasta"
  4. Haz clic en "Buscar"
  5. Verás solo las notas de ese estudiante en ese período
Casos de uso para filtros combinados:
  • Preparar reunión: Buscar notas del estudiante en los últimos 3 meses
  • Seguimiento periódico: Revisar notas del último trimestre de varios estudiantes
  • Análisis de caso: Ver todas las notas de un estudiante desde inicio del año
  • Auditoría interna: Revisar notas creadas en un período específico
Orden de aplicación de filtros: El sistema aplica los filtros en este orden:
  1. Primero: Filtro de permisos (según tu rol)
  2. Segundo: Filtros de fecha (desde/hasta)
  3. Tercero: Filtro de nombre de estudiante
  4. Resultado: Solo notas que cumplan TODOS los criterios

5 Interpretar los resultados

Los resultados se muestran en tarjetas organizadas con borde rojo distintivo.

Información mostrada en cada tarjeta:

  • Fotografía: Imagen del estudiante (si está disponible)
  • Nombre completo: Identificación del estudiante
  • Fecha de creación: Cuándo se registró la nota
  • Contenido de la nota: Texto completo con scroll si es extenso
  • Borde rojo: Recordatorio visual del carácter confidencial
Organización de resultados:
  • Las tarjetas se organizan en cuadrícula (hasta 3 por fila en pantalla grande)
  • Ordenadas de más reciente a más antigua
  • Cada tarjeta es independiente y tiene altura variable según el contenido
  • En dispositivos móviles se apilan en una sola columna
Lectura de notas extensas:
  • Si una nota es muy larga, verás una barra de desplazamiento en el lateral
  • Puedes hacer scroll dentro de la tarjeta sin mover el resto de la página
  • La barra de scroll es roja para mantener el tema de confidencialidad
Sobre las fotografías:
  • Si el estudiante tiene foto, se mostrará circular con borde rojo
  • Si no tiene foto, aparecerá un círculo rojo con la inicial del nombre
  • Si falla la carga de la foto, se mostrará una silueta genérica
Recordatorio crítico: Aunque la información está visible en pantalla:
  • NO tomes capturas de pantalla
  • NO copies el texto a otros programas
  • NO dejes la pantalla visible sin supervisión
  • CIERRA la búsqueda cuando termines de consultar

6 Limpiar búsqueda y realizar nueva consulta

Después de consultar, puedes limpiar los filtros para hacer una nueva búsqueda.

Para limpiar y buscar nuevamente:

  1. Haz clic en el botón "Limpiar" (con ícono de X circular)
  2. Se borrarán todos los campos de búsqueda
  3. Desaparecerán los resultados mostrados
  4. Puedes ingresar nuevos criterios de búsqueda
  5. Haz clic en "Buscar" nuevamente
Qué hace el botón "Limpiar":
  • Vacía el campo "Estudiante"
  • Vacía el campo "Desde"
  • Vacía el campo "Hasta"
  • Elimina los resultados de la pantalla
  • NO cierra tu sesión ni sale de la función
Buena práctica: Limpia la búsqueda después de cada consulta para evitar:
  • Dejar información visible innecesariamente
  • Confusión con búsquedas anteriores
  • Resultados no deseados en siguientes búsquedas
Al terminar de usar la función:
  • Haz clic en "Limpiar" para borrar resultados
  • Cierra la pestaña o navega a otra sección
  • Si dejas el computador, cierra sesión del sistema

Solución de problemas comunes

No veo ninguna nota al buscar
  • Verifica que hayas hecho clic en el botón "Buscar"
  • Si no tienes rol especial, solo verás tus propias notas
  • Confirma que existan notas que coincidan con tus criterios
  • Revisa los filtros de fecha (pueden ser muy restrictivos)
  • Intenta buscar sin filtros para ver si hay notas disponibles
Veo muy pocas notas aunque busco sin filtros
  • Si NO tienes rol especial (Estrategas, Directivos, EAE, Super Admin):
  • El sistema automáticamente filtra para mostrarte SOLO tus notas
  • Esto es correcto y no es un error
  • Si necesitas ver notas de otros, consulta con administración sobre roles
La búsqueda por nombre no encuentra al estudiante
  • Verifica la ortografía del nombre
  • Intenta solo con el apellido
  • Intenta solo con el nombre
  • Prueba con menos letras (más general)
  • Confirma que existan notas de ese estudiante
Los filtros de fecha no funcionan
  • Verifica que "Desde" sea anterior a "Hasta"
  • Confirma el formato de fecha de tu navegador
  • Intenta con un rango más amplio primero
  • Si el problema persiste, deja las fechas vacías
No aparecen las fotografías
  • Algunos estudiantes pueden no tener foto registrada
  • Verás una inicial en círculo rojo en su lugar
  • Esto no afecta la funcionalidad de consulta
  • La información importante es el contenido de la nota
El texto de la nota está cortado
  • Las notas largas tienen scroll interno
  • Busca una barra de desplazamiento roja en el lateral derecho
  • Haz scroll dentro del cuadro de la nota
  • También puedes usar la rueda del mouse sobre la nota
El botón "Limpiar" no funciona
  • Verifica que hayas hecho clic directamente en el botón
  • Si no responde, recarga la página (F5)
  • Puedes borrar manualmente cada campo si es necesario
La búsqueda es muy lenta
  • Esto puede ocurrir si hay muchas notas en el sistema
  • Verás un indicador de carga mientras busca
  • Usa filtros de fecha para reducir el volumen de búsqueda
  • Especifica nombre de estudiante para búsquedas más rápidas

Buenas prácticas de confidencialidad

  • Accede a la función solo cuando tengas necesidad profesional específica
  • Usa búsquedas específicas (no "curiosear" sin motivo)
  • Asegúrate de que nadie no autorizado pueda ver tu pantalla
  • Limpia los resultados después de cada consulta
  • Cierra sesión si te alejas del computador
  • NO discutas contenido de notas en lugares públicos
  • NO menciones información de notas en comunicaciones no cifradas
  • Usa la información solo para el propósito autorizado
  • Si detectas una violación de confidencialidad, repórtala inmediatamente
  • Documenta el motivo de tus consultas si tu institución lo requiere
  • Respeta que otros usuarios pueden ver o no ver las mismas notas que tú
  • NO intentes acceder a información fuera de tus permisos
Prohibiciones absolutas:
  • NUNCA tomes capturas de pantalla
  • NUNCA copies y pegues el contenido
  • NUNCA compartas información con personas no autorizadas
  • NUNCA uses la información para fines personales o externos
  • NUNCA dejes la pantalla visible sin supervisión

Preguntas frecuentes

¿Puedo crear nuevas notas desde esta función?

No. Esta es una función de SOLO CONSULTA. Para crear notas confidenciales debes usar la función específica de creación en el módulo de seguimientos.

¿Cómo sé si puedo ver todas las notas o solo las mías?

El sistema determina esto automáticamente según tu rol. Si tienes uno de los roles especiales (Estrategas, Directivos docentes, EAE, Super Admin), verás todas las notas. Si no, solo las tuyas. No hay indicador visual, pero lo notarás por la cantidad de resultados.

¿Quién puede ver que estoy consultando estas notas?

El sistema NO notifica a otros usuarios cuando consultas notas. Sin embargo, los administradores del sistema podrían tener acceso a logs de auditoría si están implementados.

¿Puedo saber quién escribió cada nota?

El sistema registra el autor (email) de cada nota en la base de datos, pero esta información NO se muestra en la interfaz de consulta actual. Si necesitas saber quién escribió una nota específica, consulta con administración.

¿Las notas tienen fecha de última modificación?

No. Las notas confidenciales solo registran la fecha de creación. No hay sistema de edición posterior, por lo que solo existe la fecha original.

¿Puedo buscar por contenido de la nota?

No directamente. La búsqueda solo funciona por nombre de estudiante y fechas. No hay búsqueda por palabras clave dentro del contenido de las notas.

¿Hay límite de notas que puedo ver?

No hay límite establecido, pero por rendimiento se recomienda usar filtros cuando sea posible. Búsquedas muy amplias pueden tardar más en cargar.

¿Puedo exportar o imprimir las notas?

NO está habilitada ninguna función de exportación o impresión. Las notas deben permanecer exclusivamente en el sistema por razones de confidencialidad.

¿Qué pasa si veo información incorrecta o inapropiada en una nota?

Reporta inmediatamente al coordinador del equipo EAE o al administrador del sistema. NO intentes corregirla tú mismo desde esta función (no tiene capacidad de edición).

¿Las notas confidenciales caducan o se eliminan automáticamente?

No hay sistema automático de eliminación. Las notas permanecen en el sistema indefinidamente a menos que un administrador las elimine manualmente siguiendo protocolos institucionales.

¿Puedo usar esta información en reuniones con padres?

Depende del contenido específico y los protocolos institucionales. Las notas confidenciales son principalmente para conocimiento interno del equipo. Si necesitas compartir algo con padres, consulta primero con el equipo directivo/EAE.

¿Qué hago si accidentalmente veo información que no debería ver?

Si el sistema te mostró información (significa que tienes los permisos), pero consideras que no deberías tener acceso a ese contenido específico, cierra inmediatamente y consulta con el coordinador de tu equipo sobre protocolos.

¿Encontraste un problema o tienes una sugerencia?

Si experimentaste un error, algo no funciona como esperas, o tienes una sugerencia para mejorar el sistema, repórtalo aquí.
