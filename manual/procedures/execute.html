Ejecutar Procedimiento

Si tienes dudas o encuentras algún problema,
puedes crear un ticket desde el botón en la esquina inferior.

¿Para qué sirve esta página?

Esta página te permite iniciar un procedimiento institucional. Verás un catálogo con los procedimientos disponibles y, al seleccionar uno, se abrirá el formulario correspondiente para que lo llenes y envíes tu solicitud.

Una vez enviada la solicitud, el sistema creará automáticamente las tareas necesarias y notificará al responsable del primer paso. Podrás consultar el estado de tu solicitud en la sección "Mis Solicitudes".

Requisito: Necesitas el permiso "Ejecutar procedimientos" asignado a tu rol, o ser administrador del sistema. Solo se muestran procedimientos internos y activos.

Catálogo de Procedimientos

Al ingresar verás una cuadrícula con tarjetas, una por cada procedimiento disponible. Solo aparecen los procedimientos que están activos y tienen pasos configurados.

Cada tarjeta muestra:

  • Nombre del procedimiento con un ícono identificativo.
  • Etiqueta "Interno" que indica que requiere inicio de sesión.
  • Descripción del procedimiento (si la tiene).
  • Cantidad de pasos que tiene el flujo de trabajo.
  • Botón "Iniciar Procedimiento" para comenzar.
Buscar

Usa el campo de búsqueda en la parte superior para encontrar un procedimiento por nombre o descripción. Los resultados se filtran en tiempo real mientras escribes.

Iniciar un Procedimiento

Para iniciar un procedimiento sigue estos pasos:

  1. Busca y selecciona el procedimiento que necesitas en el catálogo.
  2. Presiona el botón "Iniciar Procedimiento" en la tarjeta correspondiente.
  3. Se abrirá una ventana con el formulario asociado al procedimiento.
  4. Completa todos los campos requeridos (marcados con asterisco rojo *).
  5. Presiona "Enviar Solicitud" para enviar.
Importante: Una vez que envíes la solicitud, no podrás modificar los datos del formulario. Asegúrate de que la información sea correcta antes de presionar "Enviar Solicitud".

El Formulario

Al presionar "Iniciar Procedimiento" se abre una ventana modal con el formulario que debes completar. El formulario es específico para cada procedimiento y puede contener distintos tipos de campos según lo que se necesite recopilar.

Tipos de campos que puedes encontrar
Texto corto

Un campo para escribir respuestas breves como nombres, códigos o datos simples. Algunos pueden tener validación de formato (email, teléfono o URL).

Texto largo

Un área de texto más grande para escribir descripciones, observaciones o justificaciones extensas.

Número

Un campo que solo acepta valores numéricos. Puede tener un valor mínimo y máximo permitido.

Fecha

Un selector de fecha con calendario para elegir un día específico.

Selección (Dropdown)

Una lista desplegable donde eliges una opción entre varias predefinidas.

Selección única (Radio buttons)

Varias opciones visibles al mismo tiempo. Solo puedes elegir una.

Casilla de verificación (Checkbox)

Una casilla que marcas o desmarcas. Se usa para aceptaciones o conformidades. Si es obligatoria, debes marcarla para poder enviar.

SI/NO

Un selector con solo dos opciones: Sí o No.

Selección desde tabla del sistema

Una lista desplegable cuyas opciones provienen de datos del sistema, como la lista de estudiantes, trabajadores, grados, cursos, áreas académicas, entre otros. Las opciones se actualizan automáticamente según los datos vigentes.

Archivo

Un campo para adjuntar un documento. Debajo del campo se indica el tamaño máximo permitido (en MB) y los tipos de archivo aceptados (por ejemplo: pdf, jpg, png). Si seleccionas un archivo que excede el tamaño o no es del tipo permitido, el sistema te avisará y no lo aceptará.

Campos con pre-llenado automático

Algunos campos se completan automáticamente con tu información y aparecen con una etiqueta gris que dice "Automático" junto a un ícono de candado. Estos campos no se pueden editar manualmente.

Los datos que pueden pre-llenarse son:

  • Tu nombre: Se toma del nombre registrado en tu cuenta.
  • Tu email: Se toma del correo asociado a tu cuenta.
  • Fecha actual: Se completa con la fecha del día en que llenas el formulario.
Nota: Si ves que tu nombre o email pre-llenado no es correcto, contacta al administrador del sistema para que actualice tu información en tu perfil de usuario.
Campos condicionales

Algunos campos solo aparecen cuando seleccionas una opción específica en otro campo. Por ejemplo, un campo "Dirección de recogida" puede aparecer solo si seleccionas "Sí" en el campo "¿Requiere transporte?".

No necesitas hacer nada especial: los campos aparecen y desaparecen automáticamente según tus selecciones. Si un campo condicional desaparece porque cambias la selección del campo del que depende, su contenido se borra automáticamente.

Adjuntar archivos

Si el formulario incluye un campo de tipo archivo, ten en cuenta:

  • El tamaño máximo y los tipos permitidos se indican debajo del campo.
  • Si seleccionas un archivo que excede el límite, el sistema te avisará y lo rechazará.
  • Si el tipo de archivo no es permitido, también será rechazado con un mensaje de aviso.
  • El archivo se sube al momento de enviar la solicitud, no antes. Mientras aparece una barra de progreso.

¿Qué sucede al enviar la solicitud?

Cuando presionas "Enviar Solicitud", el sistema realiza las siguientes acciones de forma automática:

  1. Valida los campos: Verifica que todos los campos obligatorios estén completos y que los formatos sean correctos. Si hay algún error, te muestra un mensaje indicando cuál campo tiene el problema.
  2. Crea la solicitud: Registra una nueva instancia del procedimiento con toda la información que llenaste.
  3. Guarda las respuestas: Cada campo del formulario se almacena individualmente. Los archivos adjuntos se suben en este momento.
  4. Crea la primera tarea: Asigna automáticamente una tarea al responsable del primer paso del procedimiento. Si el primer paso es de tipo paralelo, se crean múltiples tareas simultáneas.
  5. Envía notificaciones: Te envía un correo de confirmación con el número de referencia de tu solicitud, y también notifica al responsable del primer paso que tiene una nueva tarea asignada.
  6. Te redirige: Después de unos segundos, te lleva automáticamente a la página "Mis Solicitudes" donde puedes consultar el estado de tu solicitud.
Consejo: Guarda el número de referencia que aparece en el correo de confirmación. Te servirá para dar seguimiento a tu solicitud.
Si algo falla: Si ocurre un error durante el envío, el sistema te mostrará un mensaje con el detalle del problema. El botón de envío se rehabilitará para que puedas intentar de nuevo sin perder los datos que ya llenaste.

Notificaciones por Correo

Al enviar tu solicitud se generan dos correos automáticos:

Para ti (solicitante)

Recibes un correo de confirmación con el asunto "Procedimiento iniciado" que incluye el nombre del procedimiento y un número de referencia. Este correo te confirma que la solicitud fue recibida correctamente.

Para el responsable del primer paso

La persona encargada del primer paso recibe un correo con el asunto "Nueva tarea asignada" que incluye el nombre del paso, el nombre del procedimiento y la fecha límite para completarlo. Si el primer paso es de tipo paralelo, cada responsable asignado recibe su propia notificación.

Nota: Las notificaciones se envían como complemento informativo. Si por alguna razón el correo no se entrega, la solicitud y la tarea ya están creadas en el sistema. El responsable también puede ver la tarea directamente desde su panel de tareas.

Preguntas Frecuentes

¿Puedo iniciar el mismo procedimiento más de una vez?

Sí. Cada vez que inicias un procedimiento se crea una solicitud independiente. Puedes tener múltiples solicitudes del mismo procedimiento en curso simultáneamente.

¿Puedo cancelar una solicitud después de enviarla?

Desde esta página no. Para cancelar una solicitud en curso, dirígete a la sección "Mis Solicitudes" donde encontrarás la opción de cancelación si está disponible para ese procedimiento.

¿Por qué no veo ningún procedimiento en el catálogo?

Puede ocurrir por varias razones: no hay procedimientos activos configurados en el sistema, los procedimientos existentes no tienen pasos definidos, o solo hay procedimientos de tipo externo (que no aparecen aquí porque se acceden mediante enlace público). Consulta con el administrador del sistema.

¿Qué pasa si cierro la ventana del formulario sin enviar?

No se crea ninguna solicitud. Los datos que hayas escrito se pierden. Puedes volver a iniciar el procedimiento cuando lo necesites.

¿Por qué un campo aparece bloqueado y dice "Automático"?

Ese campo se completa automáticamente con tu información (nombre, email o fecha actual). No necesitas ni puedes editarlo. Si los datos mostrados no son correctos, contacta al administrador para actualizar tu perfil.

¿Cuánto tarda en procesarse mi solicitud?

La solicitud se crea de forma inmediata al presionar "Enviar Solicitud". El tiempo total del procedimiento depende de cuántos pasos tenga y de la rapidez con que cada responsable complete su tarea. Puedes consultar el avance en "Mis Solicitudes".

¿Recibiré notificaciones cuando avance mi solicitud?

Al momento de iniciar recibes el correo de confirmación. Dependiendo de cómo esté configurado el procedimiento, podrías recibir notificaciones adicionales en pasos intermedios o al finalizar. Cuando el procedimiento se complete, recibirás una notificación indicándote que tu solicitud fue resuelta.

¿Qué hago si me sale un error al adjuntar un archivo?

Verifica que el archivo no exceda el tamaño máximo indicado debajo del campo y que sea de uno de los tipos permitidos. Si el problema persiste, intenta con un archivo más pequeño o en otro formato aceptado. Si ninguna opción funciona, reporta el problema mediante un ticket.
