Llenar Formulario

Si tienes dudas o encuentras algún problema,
puedes crear un ticket desde el botón en la esquina inferior.

¿Para qué sirve esta página?

Esta página te permite llenar formularios independientes, es decir, formularios que no están asociados a un procedimiento con pasos de aprobación. Son útiles para recopilar información puntual: encuestas internas, solicitudes simples, registros de datos, entre otros.

A diferencia de "Ejecutar Procedimiento" (que inicia un flujo de trabajo con pasos y responsables), aquí simplemente llenas un formulario y envías tu respuesta. La información queda registrada de forma inmediata.

Requisito: Necesitas el permiso "Ejecutar formularios" asignado a tu rol, o ser administrador del sistema. Solo se muestran formularios configurados en modo "Solo ejecución independiente" o "Ambos".

Catálogo de Formularios

Al ingresar verás una cuadrícula con tarjetas, una por cada formulario disponible. Cada tarjeta muestra:

  • Nombre del formulario con un ícono identificativo.
  • Cantidad de campos que contiene el formulario.
  • Descripción del formulario (si la tiene).
  • Botón "Llenar" para abrir el formulario y comenzar a completarlo.
Buscar

Usa el campo de búsqueda en la parte superior para encontrar un formulario por nombre o descripción. Los resultados se filtran en tiempo real.

Cómo llenar un formulario

Sigue estos pasos:

  1. Busca el formulario que necesitas en el catálogo.
  2. Presiona el botón "Llenar" en la tarjeta correspondiente.
  3. Se abre una ventana con el formulario. Si tiene descripción, aparece en un recuadro azul en la parte superior.
  4. Completa todos los campos requeridos (marcados con asterisco rojo *).
  5. Presiona "Enviar" para guardar tu respuesta.
  6. Verás un mensaje de confirmación con un check verde y el texto "¡Formulario enviado!".
  7. Presiona "Volver a formularios" para regresar al catálogo.
Importante: La ventana del formulario está configurada para que no se cierre accidentalmente al hacer clic fuera de ella. Debes usar el botón "Cancelar" o la X para cerrarla sin enviar. Los datos no enviados se pierden al cerrar.

Tipos de campos que puedes encontrar

Texto corto

Un campo para respuestas breves. Algunos pueden tener validación de formato (email, teléfono o URL). Si es de tipo email, el sistema valida automáticamente que el formato sea correcto.

Texto largo

Un área de texto más grande para escribir descripciones, observaciones o comentarios extensos.

Número

Solo acepta valores numéricos. Puede tener un rango permitido (valor mínimo y máximo).

Fecha

Un selector de fecha con calendario para elegir un día específico.

Selección (Dropdown)

Una lista desplegable donde eliges una opción entre varias predefinidas.

Selección única (Radio buttons)

Varias opciones visibles al mismo tiempo. Solo puedes elegir una.

Casilla de verificación (Checkbox)

Una casilla que marcas o desmarcas, presentada dentro de un recuadro gris para mayor visibilidad. Se usa para aceptaciones o conformidades. Si es obligatoria, debes marcarla para poder enviar.

SI/NO

Un selector con dos opciones: Sí o No.

Selección desde tabla del sistema

Una lista desplegable cuyas opciones vienen de datos del sistema: estudiantes, trabajadores, grados, cursos, áreas académicas, secciones o familias. Las opciones se cargan automáticamente al abrir el formulario y reflejan los datos vigentes.

Archivo

Un área de carga con un ícono de nube. Haz clic sobre ella para seleccionar un archivo de tu computador. Al seleccionar un archivo válido, el área cambia a verde y muestra el nombre y tamaño del archivo. Debajo del área se indican los tipos permitidos y el tamaño máximo.

Campos con pre-llenado automático

Algunos campos se completan automáticamente con tu información al abrir el formulario. Los datos que pueden pre-llenarse son:

  • Tu nombre: Se toma del nombre registrado en tu cuenta.
  • Tu email: Se toma del correo asociado a tu cuenta.
  • Fecha actual: Se completa con la fecha del día.
Nota: A diferencia de "Ejecutar Procedimiento", en esta página los campos pre-llenados no se bloquean visualmente. Sin embargo, los datos se completan automáticamente al abrir el formulario.
Campos condicionales

Algunos campos solo aparecen cuando seleccionas una opción específica en otro campo. Funcionan de la misma manera que en "Ejecutar Procedimiento": los campos aparecen y desaparecen automáticamente según tus selecciones, y si un campo condicional desaparece, su contenido se borra.

Ejemplo: Si un campo "¿Tiene alergia alimentaria?" con opciones "Sí" y "No" controla un campo "Describa la alergia", este segundo campo solo aparecerá cuando selecciones "Sí". Si cambias a "No", el campo desaparece y la descripción se borra.
Adjuntar archivos

Si el formulario incluye campos de tipo archivo:

  • Haz clic en el área de carga (ícono de nube) para abrir el selector de archivos.
  • El área muestra los tipos permitidos y el tamaño máximo.
  • Al seleccionar un archivo válido, el área se pone verde y muestra el nombre y tamaño.
  • Si el archivo excede el tamaño máximo, se muestra una alerta y el archivo se rechaza.
  • El archivo se sube al servidor cuando presionas "Enviar", no antes.
Validación al enviar

Al presionar "Enviar", el sistema verifica que todos los campos obligatorios estén completos. Si falta alguno:

  1. El campo incompleto se resalta con un borde rojo.
  2. La vista se desplaza automáticamente hasta el primer campo con error.
  3. Se muestra un mensaje pidiendo completar los campos obligatorios.

Solo los campos visibles se validan. Si un campo condicional está oculto porque la condición no se cumple, no se exige aunque esté marcado como obligatorio.

¿Qué sucede al enviar?

Cuando presionas "Enviar", el sistema realiza estas acciones automáticamente:

  1. Valida los campos: Verifica que los campos obligatorios visibles estén completos. Si hay errores, los señala sin enviar.
  2. Recopila las respuestas: Recoge el valor de cada campo, incluyendo las etiquetas de las opciones seleccionadas (no códigos internos).
  3. Sube los archivos: Si hay campos de archivo con documentos seleccionados, se suben al servidor en este momento.
  4. Crea el registro: Guarda la respuesta completa en el sistema. A diferencia de un procedimiento, el registro queda inmediatamente en estado "Completado" porque no hay pasos adicionales que seguir.
  5. Guarda cada respuesta: Almacena individualmente la respuesta de cada campo para permitir consultas y análisis posteriores.
  6. Envía notificación: Si el formulario tiene configurado un email de notificación, se envía un correo a esa dirección informando que alguien completó el formulario, con el nombre del formulario, quién lo llenó, la fecha y el ID del registro.
  7. Muestra confirmación: Se cierra el formulario y aparece una ventana con un check verde y el mensaje "¡Formulario enviado!".
Si algo falla: Si ocurre un error durante el envío, se muestra un mensaje de alerta y el botón de envío se rehabilita para que puedas intentar de nuevo. Los datos que escribiste se conservan en el formulario.

Diferencia con "Ejecutar Procedimiento"

Ambas páginas permiten llenar formularios, pero tienen propósitos distintos:

Ejecutar Procedimiento

Inicia un flujo de trabajo con pasos, responsables y plazos. Tu solicitud pasa por varias etapas de revisión o acción antes de completarse. Genera tareas para otras personas y puedes hacer seguimiento desde "Mis Solicitudes".

Llenar Formulario (esta página)

Envía una respuesta puntual que queda registrada de inmediato como completada. No genera tareas ni pasos adicionales. Es como llenar una encuesta o un registro: envías la información y listo.

Nota: Un mismo formulario puede estar configurado para usarse en ambos modos. Si ves el mismo formulario aquí y en "Ejecutar Procedimiento", la diferencia es que aquí no se activa ningún flujo de trabajo.

Notificaciones por Correo

Algunos formularios tienen configurado un email de notificación. Cuando esto ocurre, al enviar tu respuesta se envía automáticamente un correo a esa dirección con:

  • Nombre del formulario que se completó.
  • Tu nombre y correo electrónico.
  • Fecha y hora del envío.
  • ID del registro para referencia.

Tú como usuario no recibes un correo de confirmación. La notificación va dirigida a la persona responsable del formulario para que sepa que hay una nueva respuesta.

Nota: Si la notificación no se entrega por algún motivo técnico, tu respuesta queda registrada de todas formas. El envío del correo es complementario.

Preguntas Frecuentes

¿Puedo llenar el mismo formulario más de una vez?

Sí. Cada vez que llenas y envías un formulario se crea un registro independiente. Puedes enviar tantas respuestas como necesites.

¿Puedo editar una respuesta después de enviarla?

No. Una vez enviada, la respuesta queda registrada y no se puede modificar. Si cometiste un error, puedes enviar una nueva respuesta con los datos correctos y notificar al responsable del formulario.

¿Dónde puedo ver las respuestas que he enviado?

Las respuestas de formularios independientes se registran como instancias completadas. Si tienes acceso a "Mis Solicitudes" o "Consultar Registros", podrás verlas allí como registros con estado "Completado".

¿Por qué no veo ningún formulario en el catálogo?

Solo aparecen formularios configurados en modo "Solo ejecución independiente" o "Ambos". Si no hay ninguno disponible, puede ser que todos los formularios estén configurados para usarse únicamente dentro de procedimientos. Consulta con el administrador.

¿Qué pasa si cierro la ventana del formulario sin enviar?

No se guarda nada. Los datos que hayas escrito se pierden. Puedes volver a abrir el formulario y llenarlo desde cero cuando lo necesites.

¿Recibiré un correo de confirmación al enviar?

No. A diferencia de "Ejecutar Procedimiento", al llenar un formulario independiente no se envía correo de confirmación al usuario. La confirmación es la pantalla con el check verde que aparece después de enviar. Si el formulario tiene email de notificación configurado, ese correo va al responsable, no a ti.

¿Un campo de tipo "Selección desde tabla del sistema" muestra opciones desactualizadas?

Las opciones se cargan en tiempo real cada vez que abres el formulario. Si alguien acaba de agregar un trabajador o un estudiante al sistema, debería aparecer inmediatamente al abrir el formulario. Si no aparece, intenta cerrar y volver a abrir.

¿Puedo adjuntar más de un archivo en un campo de archivo?

No. Cada campo de archivo acepta un solo documento. Si el formulario necesita que adjuntes varios documentos, habrá un campo de archivo separado para cada uno.
