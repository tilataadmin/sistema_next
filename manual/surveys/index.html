¿Qué es el Módulo de Encuestas?

El módulo de Encuestas es un sistema completo para crear, aplicar y analizar encuestas institucionales. Permite diseñar cuestionarios con diferentes tipos de preguntas, aplicarlos a distintos públicos y obtener reportes detallados de los resultados.

La página principal del módulo (índice) sirve como centro de navegación desde donde puedes acceder a todas las funciones disponibles según tus permisos, además de ver un resumen rápido de las métricas del módulo.

¿Quién puede acceder?
Cualquier usuario que tenga al menos un permiso relacionado con el módulo de encuestas podrá ver esta página. Sin embargo, solo verá las funciones para las que tenga permisos específicos.
Flujo típico de trabajo:
  1. Configurar escalas de medición
  2. Crear la encuesta maestra con sus secciones y preguntas
  3. Asociar segmentaciones (opcional)
  4. Crear una aplicación y abrir el período de respuestas
  5. Cerrar la aplicación y analizar resultados

Funciones del Módulo

Configuración
Gestionar Escalas

Administrar escalas de medición reutilizables (Likert, Frecuencia, Satisfacción, etc.). Las escalas definen las opciones de respuesta para las preguntas de opción múltiple.

Diseño de Encuestas
Crear Encuestas

Crear y gestionar plantillas maestras de encuestas. Define el nombre, descripción, público objetivo y configuraciones generales.

Asociar Segmentaciones

Define las variables de segmentación que aplicarán a cada encuesta (grado, género, etc.) para filtrar resultados posteriormente.

Gestionar Secciones

Organiza la encuesta en secciones temáticas. Cada sección puede tener su propia escala de medición y agrupa preguntas relacionadas.

Gestionar Preguntas

Crear y organizar las preguntas dentro de cada sección. Soporta preguntas tipo escala (con opciones) y preguntas abiertas (texto libre).

Aplicaciones
Gestionar Aplicaciones

Crear instancias de aplicación de una encuesta maestra. Cada aplicación tiene su propio período de apertura, URL única y recolecta respuestas independientes.

Análisis y Reportes
Ver Resultados

Consultar los resultados de aplicaciones cerradas. Incluye promedios por sección, distribución de respuestas, análisis de preguntas abiertas y exportación.

Comparar Aplicaciones

Comparar resultados entre diferentes aplicaciones de la misma encuesta para analizar evolución temporal y tendencias.

Dashboard de Encuestas

Vista ejecutiva consolidada con indicadores clave, gráficos de tendencias, estado operativo y actividad reciente del módulo.

Microencuestas
Microencuestas

Encuestas rápidas de satisfacción para tablets en puntos de servicio (cafetería, enfermería, biblioteca). Incluye kiosko a pantalla completa, sets de imágenes personalizables y dashboard de resultados con gráficos.

Guía Paso a Paso

1 Acceder al módulo

Desde el Dashboard principal de SchoolNet, haz clic en la tarjeta "Encuestas" o accede desde el menú de navegación.

2 Revisar las métricas (opcional)

Si deseas ver el resumen numérico del módulo, haz clic en "Cargar Métricas". Verás:

  • Cuántas encuestas maestras existen
  • Cuántas aplicaciones están abiertas
  • El total de respuestas recibidas
  • Las escalas disponibles
3 Identificar las funciones disponibles

Observa el grid de funciones. Solo verás las tarjetas para las que tienes permisos. Si no ves alguna función que necesitas, contacta al administrador para solicitar el permiso.

4 Navegar a la función deseada

Haz clic en la tarjeta de la función que necesitas. Cada función tiene su propia página con instrucciones específicas.

Consejo: Si es tu primera vez usando el módulo, te recomendamos seguir este orden: Escalas → Encuestas → Secciones → Preguntas → Aplicaciones.

Funcionalidades Adicionales

Sistema de Permisos

El módulo utiliza un sistema de permisos granular. Cada función tiene su propio permiso que debe estar asignado al rol del usuario:

  • Gestionar escalas: Crear y editar escalas de medición
  • Crear encuestas: Gestionar encuestas maestras
  • Asociar segmentaciones: Configurar variables de segmentación
  • Gestionar secciones de encuestas: Organizar secciones
  • Gestionar preguntas: Crear y editar preguntas
  • Gestionar aplicaciones: Abrir y cerrar aplicaciones
  • Ver resultados: Acceder al análisis de resultados
  • Comparar aplicaciones: Comparar entre períodos
  • Dashboard de encuestas: Ver vista ejecutiva
  • Gestionar microencuestas: Crear, editar y consultar microencuestas de satisfacción rápida
Carga Diferida de Métricas

Las métricas no se cargan automáticamente al entrar a la página. Esto mejora el rendimiento y evita consultas innecesarias cuando solo necesitas navegar rápidamente a una función.

Grupos Dinámicos

Si no tienes permisos para ninguna función de un grupo, ese grupo completo se oculta automáticamente para mantener la interfaz limpia y relevante.

Buenas Prácticas

  • Planifica antes de crear: Antes de crear una encuesta, define claramente el objetivo, el público objetivo y las preguntas que necesitas hacer.
  • Reutiliza escalas: Crea escalas genéricas que puedas usar en múltiples encuestas (Likert 1-5, Frecuencia, Satisfacción, etc.).
  • Organiza en secciones: Agrupa las preguntas por temas para facilitar el análisis y mejorar la experiencia del encuestado.
  • Prueba antes de aplicar: Antes de abrir una aplicación, revisa toda la encuesta para verificar que las preguntas y escalas estén correctas.
  • Usa segmentaciones: Configura segmentaciones para poder filtrar resultados por grado, género u otras variables relevantes.
  • Analiza oportunamente: Una vez cerrada una aplicación, revisa los resultados mientras la información está fresca para tomar acciones.

Preguntas Frecuentes

¿Por qué no veo todas las funciones del módulo?
Solo se muestran las funciones para las que tienes permisos asignados. Si necesitas acceso a una función específica, contacta al administrador del sistema para que asigne el permiso correspondiente a tu rol.
¿Cuál es la diferencia entre "Encuesta Maestra" y "Aplicación"?
La Encuesta Maestra es la plantilla o diseño de la encuesta (preguntas, secciones, escalas). La Aplicación es una instancia específica de esa plantilla con un período de apertura definido. Puedes aplicar la misma encuesta maestra múltiples veces (por ejemplo, cada trimestre).
¿Puedo editar una encuesta después de que tiene aplicaciones?
Puedes editar la encuesta maestra, pero ten cuidado: los cambios en preguntas o escalas podrían afectar la consistencia de los resultados entre aplicaciones. Se recomienda no modificar encuestas que ya tienen aplicaciones con respuestas.
¿Qué es una "Escala de medición"?
Es un conjunto de opciones de respuesta predefinidas. Por ejemplo, una escala Likert de 5 puntos tiene opciones como "Totalmente en desacuerdo" (1) hasta "Totalmente de acuerdo" (5). Las escalas se configuran una vez y se reutilizan en múltiples encuestas.
¿Cómo funciona la segmentación?
La segmentación permite recolectar información adicional del encuestado (como grado, género, etc.) que luego se usa para filtrar y comparar resultados. Por ejemplo, puedes ver los resultados solo de estudiantes de 10° grado o comparar respuestas entre géneros.
¿Por qué las métricas muestran "--"?
Las métricas usan carga diferida para mejorar el rendimiento. Haz clic en "Cargar Métricas" para ver los valores actualizados. Una vez cargadas, el botón cambia a "Actualizar Métricas".
¿Qué público objetivo puedo seleccionar para una encuesta?
El sistema soporta cuatro tipos de público:
  • Estudiantes: Encuestas para alumnos
  • Familias: Encuestas para padres/acudientes
  • Trabajadores: Encuestas para personal de la institución
  • Mixtas: Encuestas que combinan varios públicos
¿Las encuestas son anónimas?
Sí, las respuestas son anónimas. El sistema registra datos de segmentación (si están configurados) pero no identifica individualmente a cada encuestado. Esto fomenta respuestas honestas y protege la privacidad.
¿Puedo ver los resultados mientras la encuesta está abierta?
No. La función "Ver Resultados" solo muestra aplicaciones cerradas. Esto evita que los resultados parciales influyan en decisiones prematuras y garantiza que se analice el conjunto completo de respuestas.
¿Qué son las microencuestas?
Son encuestas de satisfacción rápida diseñadas para tablets en puntos de servicio como cafetería, enfermería o biblioteca. Muestran tres imágenes grandes (positiva, neutra, negativa) a pantalla completa. Las respuestas son anónimas, no requieren inicio de sesión, y los resultados se consultan desde un dashboard con gráficos y filtros por rango de fechas.
¿En qué orden debo configurar una encuesta nueva?
El orden recomendado es:
  1. Crear escalas de medición (si no existen las que necesitas)
  2. Crear la encuesta maestra
  3. Asociar segmentaciones (si aplica)
  4. Crear secciones
  5. Crear preguntas en cada sección
  6. Crear la aplicación cuando todo esté listo
